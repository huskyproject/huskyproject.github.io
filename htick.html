What is it?

htick is a file ticker.

What does it need?

Operating systems:
  • All supported platforms.
Dependencies:
  • htick-current requires fidoconf-current, smapi-current and huskybse-current from the same day
  • htick-0.9-stable requires fidoconf-0.10-stable, smapi-1.6.4-stable and huskybse-0.1

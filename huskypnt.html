What is it?

HuskyPnt is a point-package based on husky-Software. After answering a few questions (most can just be skipped, there are reasonable defaults) it will compile and install the libraries (smapi, fidoconf) and programs and generate a working config.
It is intended to support almost any platform and (natural) language and will be multi-user capable, but currently it only runs under Linux with german or english questions (the config remarks are in german) and for a single user.

What does it need?

Operating systems:
  • Linux
The following husky packages compressed with zip (pkzip or info zip, not GNU gzip!):
  • fidoconf (fidoconf.zip)
  • gff (gff.zip)
  • hpt (hpt.zip)
  • husky-common (huskycom.zip)
  • ifcico (ifcico.zip)
  • mpost (mpost.zip)
  • msged (msged.zip)
  • scripts (scripts.zip, contained in huskypnt)
  • smapi (smapi.zip)
  • sqpack (sqpack.zip)
Other files:
  • current nodelist
  • current german pointlist (points24)

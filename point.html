What do I need for a point?

Information about what to download, how to compile the software and where to find documentation can be found inside the huskybse package (see in the Basic Files category).

Required      Optional
huskybase v1.4 Base Package      msged1 Message Editor
smapi v2.4 Message Base library      htick File Ticker
fidoconf v1.4 Husky Config library         
hpt v1.4 Highly Portal Tosser         
    Notes:
  1. You can use any editor that can access the standard message base files; however, msged is configured to work with the standard Husky set up.

Download Husky!

To get the stable Husky versions, you can browse to the individual entries of each program (see over in the navigation bar), or you can directly access our project download page at  http://sourceforge.net/project/showfiles.php?group_id=1332.

If you are using Debian GNU/Linux take a look at our  Debian Page. You could install the Husky programs using the normal and easy Debian way (aptitude, apt-get, dselect,...).

Current Husky is available from the Husky project CVS repository. Do not use it without a good reason.

If you are using BeOS take a look at  Walther: Fidonet for BeOS page.

Download Debian-Packages for Husky!

Husky Debian page


For Husky-stable go here: stable
This binary packages build for Debian stable (woody).

For apt-get add in your /etc/apt/sources.list
deb http://husky.sourceforge.net/debian stable main

For build Husky-stable on Debian unstable system add in /etc/apt/sources.list
deb-src http://husky.sourceforge.net/debian stable main
and use apt-src package



For Husky-testing go here: testing
This binary packages build for Debian testing (sarge).

For apt-get add in your /etc/apt/sources.list
deb http://husky.sourceforge.net/debian testing main

For build Husky-testing on Debian stable system add in /etc/apt/sources.list
deb-src http://husky.sourceforge.net/debian testing main
and use apt-get -b source <package>

What is it?

FreeFidoMessageAssistant is a tracker which has some netmgr features and some new features.

What does it need?

Operating systems:
  • Linux
Dependencies:
  • ffma-current requires fidoconf-current, smapi-current from the same day and fpc-Pacal 1.00 (or better)
  • ffma 0.08.00 requires fidoconf-0.10-stable, smapi-1.6.4-stable and fpc-Pascal 0.99.15 (or better)

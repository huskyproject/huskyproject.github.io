What is it?

nltools contains tools for keeping your nodelists up to date, and for compiling a fidouser.lst file (but not a V7 index).

What does it need?

Operating systems:
  • All supported platforms.
Dependencies:
  • nltools-current requires fidoconf-current and huskybse-current from the same day
  • nltools-1.0 requires fidoconfig 0.9
  • nltools-1.1-stable requires fidoconfig 0.10-stable

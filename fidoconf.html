What is it?

fidoconf is a library used to acess the fidoconfig style configuration file. This file keeps the settings for most of the husky projects. The package also contains some utilities for verifying the configuration file syntax or for converting it to other common formats.

What does it need?

Operating systems:
  • All
Dependencies:
  • fidoconf-current requries smapi-current and huskybse-current from the same day
  • fidoconfig 0.10-stable requires smapi 1.6.4-stable and huskybse-0.1
  • fidoconfig 0.9 requires smapi 1.6.2

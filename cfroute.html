What is it?

cfroute is not actually a tracker, but a Binkley outbound style netmail router / packer. It does not support the fidoconfig file format.

What does it need?

Operating systems:
  • Linux/i386, FreeBSD/i386, OS/2, Win32, DOS.
  • Presently does not work on big-endian systems, on 64 bit systems, or an systems that do not feature structure packing.
Dependencies:
  • cfroute-current requires smapi-current and huskybse-current from the same day
  • cfroute 0.95a requires smapi 1.6.2
